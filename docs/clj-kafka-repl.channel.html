clj-kafka-repl.channel

core.async based functions for use with kafka/consumer-chan.

atom-sink

(atom-sink a & opts)

Creates a new sink to a specified atom initialised with [].

close!

(close! ch)

Closes the specified channel/consumer-channel.

closed?

(closed? ch)

Returns flag indicating whether the specified channel is closed.

file-sink

(file-sink f & {:keys [printer], :or {printer clojure.pprint/pprint}, :as opts})

Creates a new sink to a specified file path. Writes all messages received to the sink using pprint (can be overridden with the printer option).

poll!

(poll! {:keys [channel]})

Reads off the next value (if any) from the specified channel.

progress

(progress {:keys [progress-fn]})

Gives an indication of the progress of the given tracked channel on its partitions.

skip!

(skip! {:keys [channel]} n)

Consumes and skips n items from the consumer channel.

to-atom

(to-atom ch a & opts)

Appends all messages received in the channel to the specified atom using conj. This means that the atom should be initialised with an empty collection - e.g. (atom [])

to-file

(to-file ch f & opts)

Writes all messages received in the channel to the specified file path using pprint. To use your own pretty printing function, use (to consumer-channel f :printer <your print function>).

to-sinks!

(to-sinks! {:keys [channel]} sinks & {:keys [close-sinks?], :or {close-sinks? true}})

Starts sending data from the specified input channel to the specified sinks.

to-stdout

(to-stdout ch & opts)

Writes all messages received in the channel to stdout.

writer-sink

(writer-sink writer & {:keys [printer], :or {printer (fn* [p1__9507#] (puget/pprint p1__9507# {:print-color true}))}, :as opts})

Creates a new sink to a specified PrintWriter - e.g. out for stdout.
